# House Rent Prediction

## Overview
This project analyzes house rent data and predicts rental prices using Machine Learning techniques.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Features
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization
- Feature Engineering
- House Rent Prediction

## Models Used
- Linear Regression
- Random Forest
- XGBoost
